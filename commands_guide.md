# 📜 دليل أوامر البوت الشامل

هذا الملف يحتوي على جميع الأوامر المتوفرة في البوت وكيفية استخدامها بشكل صحيح.

---

## 🪙 نظام الكوينات (Coins System)

| الأمر | الوصف | الفئة |
| :--- | :--- | :--- |
| `/coins` | عرض رصيدك الحالي بصورة احترافية. | الجميع |
| `/coins user:@عضو` | عرض رصيد عضو آخر (للمعلمين والأونر). | الإدارة |
| `/add-coins user:@عضو amount:<عدد>` | إضافة كوينات لرصيد العضو. | الإدارة |
| `/remove-coins user:@عضو amount:<عدد>` | خصم كوينات من رصيد العضو. | الإدارة |
| `/reset-coins user:[@عضو]` | تصفير رصيد عضو أو السيرفر بالكامل. | الإدارة |

---

## 🛡️ أوامر الإدارة والرقابة (Moderation)

| الأمر | الوصف | ملاحظات |
| :--- | :--- | :--- |
| `ورن` أو `warn @عضو <سبب>` | إعطاء تحذير رسمي للعضو. | (3 تحذيرات = تايم أوت تلقائي 5 ساعات) |
| `انورن` أو `unwarn @عضو #الرقم` | إزالة تحذير محدد باستخدام رقمه. | مثال: `انورن @عضو #1` |
| `كيك` أو `kick @عضو [سبب]` | طرد العضو من السيرفر. | مخصص للأونر فقط |
| `تايم` أو `time @عضو <مدة> <سبب>` | إسكات العضو مؤقتاً. | الوحدات: `m` (دقيقة), `h` (ساعة), `d` (يوم) |
| `انتايم` أو `untime @عضو` | فك الإسكات عن العضو. | مخصص للأونر فقط |
| `ban @عضو [سبب]` | حظر العضو نهائياً من السيرفر. | يتم إرسال نبيه في الخاص واللوجات |
| `unban <ID> [سبب]` | فك الحظر عن عضو باستخدام الآيدي. | |
| `مسح` أو `mas7 <عدد>` | مسح عدد محدد من الرسائل. | الحد الأقصى 100 رسالة |
| `سمي` أو `nickname @عضو <الاسم>` | تغيير لقب العضو في السيرفر. | |

## 🔊 أنظمة تلقائية (Automatic Systems)

### 1. الغرف الخاصة (Private VC)
- **كيفية البدء**: انضم لقناة `Join to Create`.
- **التحكم**: ستظهر لك لوحة تحكم في غرفتك تتيح لك:
    - تغيير الاسم والعدد.
    - التحكم في الخصوصية (قفل، إخفاء).
    - نظام الثقة (Trust) والحظر (Block).

### 2. نظام التذاكر (Tickets)
- **كيفية البدء**: توجه لقناة الدعم الفني واضغط على زر إنشاء التذكرة.
- **المميزات**: خصوصية تامة، إشعارات خاصة عند الرد أو الإغلاق، ولوجات مفصلة للإدارة.

---

## 💡 ملاحظات هامة:
- جميع أوامر الكوينات أصبحت الآن أوامر تفاعلية (Slash Commands) وتظهر فقط للأشخاص المسموح لهم.
- الأوامر التي تبدأ بـ `!` هي أوامر كتابية (Prefix Commands).
- الأوامر التي تبدأ بـ `/` هي أوامر تفاعلية (Slash Commands).
- جميع العمليات الإدارية يتم تسجيلها في قنوات اللوجات المخصصة لضمان الشفافية.
- في أوامر التايم-أوت، إذا وضعت مدة أكثر من 28 يوماً، سيقوم البوت تلقائياً بتثبيتها عند 28 يوماً (أقصى حد مسموح).
